前缀和优化

2020 ICPC Latin American F. Fabricating Sculptures

https://codeforces.com/gym/102428/problem/F

  • 题意

    用相同的方格堆格子. 要求恰好有 nnn 列, 并且不会存雨. 即不会有一列左右两边都存在比它高的列.

    求总方案数.

  • 分析

    一开始按照列从左往右 dp. 令 dp[i][j][k][0/1]dp[i][j][k][0/1]dp[i][j][k][0/1] 表示到第 iii 列, 总共用了 jjj 个格子, 第 iii 列放 kkk 个格子, 之前是递减, 递增的方案数.

    按照题意等价于说, 格子只能是单峰的. 于是可以得到转移方程.

    dp[i][j][k][1]=∑l=0kdp[i−1][j−k][l][1]dp[i][j][k][0]=∑l=kj−kdp[i−1][j−k][l][0]+∑l=k+1j−kdp[i−1][j−k][l][1]dp[i][j][k][1] = \sum_{l = 0}^{k} dp[i - 1][j - k][l][1] \\ dp[i][j][k][0] = \sum_{l = k}^{j - k} dp[i - 1][j - k][l][0] + \sum_{l = k + 1}^{j - k} dp[i - 1][j - k][l][1] dp[i][j][k][1]=l=0∑k​dp[i−1][j−k][l][1]dp[i][j][k][0]=l=k∑j−k​dp[i−1][j−k][l][0]+l=k+1∑j−k​dp[i−1][j−k][l][1]

    直接算是 O(n4)O(n^4)O(n4), 用前缀和优化一下是 O(n3)O(n^3)O(n3). tle.

    😿

    换一种思路, 按照一层一层从下往上 dp. 可以发现每一层放的方块必须是连续的.

    令 dp[i][j]dp[i][j]dp[i][j] 表示下面一层有 iii 个格子, 还剩 jjj 个格子可以放. 枚举这一层放的格子数量.

    dp[i][j]=∑k=1min(i,j)(i−k+1)dp[k][j−k]dp[i][j] = \sum_{k = 1}^{min(i,j)} (i - k + 1) dp[k][j - k] dp[i][j]=k=1∑min(i,j)​(i−k+1)dp[k][j−k]

    可以化简一下

    dp[i][j]=∑k=1min(i,j)(i−1−k+1)dp[k][j−k]+∑k=1min(i,j)dp[k][j−k]dp[i][j] = \sum_{k = 1}^{min(i,j)} (i - 1 - k + 1) dp[k][j - k] + \sum_{k = 1}^{min(i,j)} dp[k][j - k] dp[i][j]=k=1∑min(i,j)​(i−1−k+1)dp[k][j−k]+k=1∑min(i,j)​dp[k][j−k]

    分类讨论.

    1. i>ji > ji>j

      dp[i][j]=∑k=1j(i−1−k+1)dp[k][j−k]+∑k=1min(i,j)dp[k][j−k]dp[i][j] = \sum_{k = 1}^j (i - 1 - k + 1) dp[k][j - k] + \sum_{k = 1}^{min(i,j)} dp[k][j - k] dp[i][j]=k=1∑j​(i−1−k+1)dp[k][j−k]+k=1∑min(i,j)​dp[k][j−k]

    2. i≤ji\le ji≤j

      dp[i][j]=∑k=1i(i−1−k+1)dp[k][j−k]+∑k=1min(i,j)dp[k][j−k]dp[i][j] = \sum_{k = 1}^i(i - 1 - k + 1) dp[k][j - k] + \sum_{k = 1}^{min(i,j)} dp[k][j - k] dp[i][j]=k=1∑i​(i−1−k+1)dp[k][j−k]+k=1∑min(i,j)​dp[k][j−k]

      k=ik = ik=i 的时候左边是 000. 所以

      dp[i][j]=∑k=1i−1(i−1−k+1)dp[k][j−k]+∑k=1min(i,j)dp[k][j−k]dp[i][j] = \sum_{k = 1}^{i-1}(i - 1 - k + 1) dp[k][j - k] + \sum_{k = 1}^{min(i,j)} dp[k][j - k] dp[i][j]=k=1∑i−1​(i−1−k+1)dp[k][j−k]+k=1∑min(i,j)​dp[k][j−k]

    把两部分合起来得到

    dp[i][j]=dp[i−1][j]+∑k=1min(i,j)dp[k][j−k]dp[i][j] = dp[i - 1][j] + \sum_{k = 1}^{min(i,j)}dp[k][j - k] dp[i][j]=dp[i−1][j]+k=1∑min(i,j)​dp[k][j−k]

    实际上可以直接得到这个转移方程. 考虑下一层的 iii 个格子的第一个格子放不放.

    如果不放就是 dp[i−1][j]dp[i - 1][j]dp[i−1][j]. 如果放枚举放到哪.

    直接朴素算还是O(n3)O(n^3)O(n3) 的, 把转移表画出来

    红点从蓝点转移过来. 所有的蓝点就是一个对角线. 从上往下dp 的时候顺便维护一个对角线的和. 时间复杂度就降到了O(n2)O(n^2)O(n2).
