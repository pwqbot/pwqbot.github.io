Codeforces 1543 - D2. RPD and Rap Sheet (Hard Version)

https://codeforces.com/contest/1543/problem/D1

  • 题意

    自适应猜密码.

    用 nnn 次去猜一个 [0,n−1][0,n-1][0,n−1] 范围内的一个数 xxx, 每猜一次之后 xxx 会变成zzz 满足. x⊕kz=yx \oplus_{k} z = yx⊕k​z=y.

    其中⊕k\oplus_{k}⊕k​ 是基数为 kkk 的异或.

  • 分析

    题解两种做法. 题目不是让我们猜出原来的 xxx, 而是猜出变化之后的密码.

    我们可以给出一个模式, 在第 xxx 次猜测之后让第 x+1x + 1x+1 次猜测的模式等于变化后的 xxx.

    观察这里z=y⊖kxz = y \ominus_{k} xz=y⊖k​x, 其中按照 kkk 为基数每一位

    y⊖kx=(y−x) mod ky \ominus_{k} x = (y - x) \bmod k y⊖k​x=(y−x)modk

    一个重要的性质

    (a⊖kb)⊖k(a⊖kc)=c⊖kb(a \ominus_{k} b) \ominus_{k} (a \ominus_{k} c) = c \ominus_{k} b (a⊖k​b)⊖k​(a⊖k​c)=c⊖k​b

    • 解法1

      我们可以这样构造.

      0,0⊖1,2⊖1,2⊖30, 0 \ominus 1, 2 \ominus 1, 2 \ominus3 0,0⊖1,2⊖1,2⊖3

      这样 xxx 会是

      0⊖x,x⊖1,2⊖x,x⊖30 \ominus x, x \ominus1, 2 \ominus x, x \ominus 3 0⊖x,x⊖1,2⊖x,x⊖3

      一种通用的解法.

      构造

      qi=qi−1⊖qi−2⊖qi−3⋯⊖q1⊖(i−1)q_{i} = q_{i - 1} \ominus q_{i - 2} \ominus q_{i - 3} \dots \ominus q_{1} \ominus (i - 1) qi​=qi−1​⊖qi−2​⊖qi−3​⋯⊖q1​⊖(i−1)

      但是⊖\ominus⊖ 这个东西没有结合律或者交换律, 这样算要 n2n^2n2 的.

      q1=0q2=q1⊖1q3=q2⊖q1⊖2=(q2⊖q1)⊕2q4=q3⊖q2⊖q1⊖3=(q3⊖q2⊖q1)⊖3q_{1} = 0 \\ q_{2} = q_{1} \ominus 1 \\q_{3} = q_{2} \ominus q_{1} \ominus 2 = (q_{2} \ominus q_{1}) \oplus 2 \\ q_{4} = q_{3} \ominus q_{2} \ominus q_{1} \ominus 3 = (q_{3} \ominus q_{2} \ominus q_{1}) \ominus 3 q1​=0q2​=q1​⊖1q3​=q2​⊖q1​⊖2=(q2​⊖q1​)⊕2q4​=q3​⊖q2​⊖q1​⊖3=(q3​⊖q2​⊖q1​)⊖3

Codeforces 1543 - E. The Final Pursuit

https://codeforces.com/contest/1543/problem/E

  • 题意

  • 分析

    参考资料 https://www.youtube.com/watch?v=X8jsijhllIA.

    第一部分还原排列.

    注意超立方体, 相邻节点只改变 1bit.

    先随便指定一个0. 然后可以得到原来的超立方体的 1,2,4,8,....1,2,4,8, ....1,2,4,8,.... 对应的.

    然后一个点只要相邻两个点确定了, 它就是确定的.

    第二部分涂色.

    条件是nnn 必须是 222 的幂次, 因为所有颜色的个数都要相同. 所以 2n2^{n}2n 必须是 nnn 的倍数.

    一个小性质可以记一下.

    当 nnn 是 222 的幂次的时候. 对一个小于 nnn 的整数xxx, 其异或[0,n−1][0,n-1][0,n−1] 的所有值, 是[0,n−1][0,n-1][0,n−1] 的一个排列.

    证明也很简单. 因为异或的值都不同, 并且不会超过 nnn.

    对节点x=bn−1bn−2…b0x = b_{n - 1}b_{n -2} \dots b_{0}x=bn−1​bn−2​…b0​

    令颜色 c=⊕i=0n−1biic = \oplus_{i = 0}^{n - 1} b_{i}ic=⊕i=0n−1​bi​i.

    这样改变它的一个bitbitbit , 颜色 ccc 就相当于异或 iii. 根据上面那个性质, 其邻居节点的颜色是都不同的.
