与/或 取 max

  • 分析

    对每个区间维护一个 same 值. 第 iii 位为 111 表示这个区间的数第 iii 位全是 111 或者 000.

    因为与一个数 xxx, 只需考虑 xxx 中为 000 的位. 如果 xxx 的所有为 000 的位对应的 same 都是 111. 那么与上 xxx 之后, 区间所有数的大小关系没有改变.

    或一个数 xxx 也是同理.

    • pushup

    最大值易更新.

    父亲的 same 为 111 需要两个儿子的 same 都为 111 且对应二进制位相同.

    • pushdn

      这里 same 不用 pushdn, 因为它只会在 pushup 更新.

      最大值的pushdn 其实可以通过懒惰标记简单的完成. 但这题也可以不用懒惰标记.

      对父亲的懒惰修改只会影响 same 为1的部分.
