轻重链剖分

代码

第一次dfs预处理.

  1. 父节点
  2. 重儿子
  3. 子树大小
  4. 节点深度

第二次dfs得到每条链

性质

每条路径只会包含 log(n)log(n)log(n) 条重链.

因为走一条轻边的时候, 所在子树大小至少除以2.

基本应用

  • 求LCA

    链顶端低的结点一直往上跳就行了.

  • 路径修改

    给一条路径加上的所有点加上z. 和lca差不多的思路, 就是两个点一直往上跳, 直到lca

路径查询也是这个思路.

然后还要子树修改, 子树查询, 就是和原来的dfs一样的.

因为轻重链就是一种特殊的dfs序而已.

基础

luogu P2486 染色

https://www.luogu.com.cn/problem/P2486

  • 分析

    染色问题, 把区间换成了树的路径.

    树剖之后用线段树维护区间染色就行了.

    然后就是注意在跳链的时候, 要看看链之间的端点颜色是否相同.

    最重要的是最后在一条链上的时候, 左右两个 链都要比较!.

    debug 半天, 发现是把add的一个条件写成了 else if....
