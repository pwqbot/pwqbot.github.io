vscode 快捷键

调试

  • 调试控制台 Ctrl + Shift + y

  • 查看函数定义 F12

  • 切换断点 F9

  • 单行注释/取消单行注释 Ctrl + /

  • 多行注释 Alt + Shift + A

操作

  • 整行向 左/右 缩进 Ctrl + [ , Ctrl + ]

  • 撤销 Ctrl + z

  • 恢复撤销 Ctrl + Shift + z

  • 删除一行 Ctrl + X

  • 下方添加一行 Ctrl + Enter

  • 上方添加一行 Ctrl + Shift + Enter

  • 复制当前行到下方 Shift + ALt + ⬇️

  • 复制当前行到上方 Shift + Alt + ⬆️

    额,上面两个有区别吗??

选择

  • 选择当前光标所在单词 Ctrl + d. 一直按会一直往下选
  • 选择整行 Ctrl + L. 一直按就一直往下选
  • 跳到本行最右边 end
  • 跳到本行最左边 home
  • 选择当前到最右边所有 Shift + end
  • 选择当前到最左边所有 Shift + home
  • 跳到文件头 Ctrl + home
  • 跳到文件尾 Ctrl + end
  • 跳到相应的括号 Ctrl + Shift + \
  • 添加光标 alt + 鼠标左键
  • 光标回到上个位置 Ctrl + u

窗口

  • 全屏 F11

  • 集成终端 Ctrl + `

  • Debug Consle Ctrl + Shift + y
