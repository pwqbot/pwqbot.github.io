多项式求逆

问题

给一个多项式F(x)F(x)F(x) , 求出一个多形式G(x)G(x)G(x), 满足F(x)×Q(x)≡1(modxn)F(x) \times Q(x) \equiv 1 \pmod{x^n}F(x)×Q(x)≡1(modxn), 系数对 mod 取模

分析

用递归的思想, 考虑已解决子问题

F(x)G(x)≡1(modx⌈n2⌉)F(x)G(x) \equiv 1\pmod{x^{\lceil \frac{n}{2} \rceil}} F(x)G(x)≡1(modx⌈2n​⌉)

因为F(x)Q(x)≡1(modxn)F(x)Q(x) \equiv 1 \pmod{x^{n}}F(x)Q(x)≡1(modxn), 也就是其次数小于 n 的项的系数都是0, 那么一定

F(x)Q(x)≡1(modx⌈n2⌉)F(x)Q(x) \equiv 1\pmod{x^{\lceil \frac{n}{2}\rceil}} F(x)Q(x)≡1(modx⌈2n​⌉)

把两个式子减一下

F(x)(Q(x)−G(x))≡0(modx⌈n2⌉)F(x)(Q(x) - G(x)) \equiv 0 \pmod{x^{\lceil \frac{n}{2}\rceil}} F(x)(Q(x)−G(x))≡0(modx⌈2n​⌉)

两边同乘F(x)F(x)F(x) 的逆元

Q(x)−G(x)≡0(modx⌈n2⌉)Q(x) - G(x) \equiv 0 \pmod{x^{\lceil \frac{n}{2}\rceil}} Q(x)−G(x)≡0(modx⌈2n​⌉)

模0, 意味着次数小于 ⌈n2⌉\lceil \frac{n}{2} \rceil⌈2n​⌉ 的项的系数都是0. 把这个式子平方后可以得到

Q2(x)−2Q(x)G(x)+G2(x)≡0(modxn)\\ Q^2(x) - 2Q(x)G(x) + G^2(x) \equiv 0 \pmod{x^{n}} Q2(x)−2Q(x)G(x)+G2(x)≡0(modxn)

这是因为次数小于 n 的项, 因子中一定有一项次数小于⌈n2⌉\lceil \frac{n}{2}\rceil⌈2n​⌉. 所以系数就为0

两边同乘F(x)F(x)F(x)

F(x)Q2(x)−2F(x)Q(x)G(x)+F(x)G2(x)≡0(modxn)Q(x)−2G(x)+F(x)G2(x)≡0(modxn)Q(x)=(2−F(x)G(x))G(x)(modxn)\\ F(x)Q^2(x) - 2F(x)Q(x)G(x) + F(x)G^2(x) \equiv 0 \pmod{x^{n}} \\ Q(x) - 2G(x) + F(x)G^2(x) \equiv 0 \pmod{x^{n}} \\ Q(x) = (2 - F(x)G(x))G(x)\pmod{x^{n}} F(x)Q2(x)−2F(x)Q(x)G(x)+F(x)G2(x)≡0(modxn)Q(x)−2G(x)+F(x)G2(x)≡0(modxn)Q(x)=(2−F(x)G(x))G(x)(modxn)

然后用NTT加速一下就行了.

T(n)=T(n2)+O(nlg(n))=O(nlg(n))T(n) = T(\frac{n}{2}) + O(nlg(n)) = O(nlg(n)) T(n)=T(2n​)+O(nlg(n))=O(nlg(n))

代码

这里用了一个技巧. 把三个乘法用一次ntt算.

为什么这样是可以的呢. 因为只要把ntt的上限开的比三个乘起来的最高的次数打就行.

这里F(x)F(x)F(x) 是 nnn 次. G(x)G(x)G(x) 只有 ⌈n+12⌉\lceil{ \frac{n + 1}{2} } \rceil⌈2n+1​⌉ 次, 三个乘起来也只有2n2n2n 次. 所以上限开到2n2n2n 就差不多了.
