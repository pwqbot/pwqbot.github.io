工具网站推荐

发表于2021-03-12|更新于2021-03-13|TOYS
|阅读量:

网页

  • 油猴脚本

    神器. 正在用以下几个

    1. AC baidu

      修理度娘

    2. 豆瓣资源下载大师

      神器… 搜资源, 整合评分票房信息.

    3. 知乎美化 + 知乎增强

      优化知乎体验.

    4. 贴吧全能助手

      浏览贴吧不用登录, 页面美化, 去广告等等.

    5. csdn优雅美化脚本

      美化某垃圾网站,去广告等.

    6. 网页限制解除

      可以复制原本不让复制的网站上的文字

  • chrome 扩展

    1. workona

      标签页整理. 非常好用, 因为我经常开一堆标签页然后被烦死. 这个插件把标签页分隔到不同的工作区, 比如分离学习和娱乐, 标签一下就没那么乱了, 拯救了我qwq!

    2. tabliss

      力推!!原来在火狐浏览器就用了. 每次打开一个新标签页会换一张非常非常好看的壁纸, 壁纸来源是unsplash等免费高质量的图片网站.

      我不喜欢在主页加一些花里胡哨的东西.

    3. 书签侧边栏

      美化chrome的书签管理, 很不错.

图片

  • 图片压缩

    1. iloveimg https://www.iloveimg.com

      把图片拖进去即可. 处理速度快, 支持 JPG, PNG 和 GIF 格式. 此外还有格式转换, 图片裁剪等功能

    2. squoosh https://squoosh.app/editor

      支持的格式更多一点, 可以自己调参数.

  • 素材网站

    1. 设计导航 http://hao.shejidaren.com/

      里面攘括了各种各样的设计素材网站, 包括图片, 图标, 配色等等.

    2. pexel https://www.pexels.com

      支持 关键字, 颜色, 大小, 方向
