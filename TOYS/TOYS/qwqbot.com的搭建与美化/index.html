qwqbot.com

为什么会搭建这个博客呢, 首先我本地就有很多学习笔记, 题解什么的, 挺想把它们整理好的. 其次这里也会记录关于生活上的兴趣爱好, 比如电影音乐啥的.

把我学会的东西, 感受到的东西, 思考过的东西都能记录下来的话, 是一件有成就感的事.

  • 学习

    1. ACM

      包括算法的学习和比赛的题解

    2. 专业课

      专业课的笔记和作业

    3. 科研

      论文的阅读笔记和科研的进展

    4. toys

      瞎捣鼓的一些东西, 比如这个网站

  • 生活

    1. 阅读

      上一次写读书笔记还是初中的语文作业.

    2. 电影

      从来没写过影评

    3. 音乐

      这里有两部份, 一个是分享一些我喜欢的歌和音乐人. 一个是 我的演奏和创作的内容.

    4. 随笔

      关于最近的几周或者是几个月.
