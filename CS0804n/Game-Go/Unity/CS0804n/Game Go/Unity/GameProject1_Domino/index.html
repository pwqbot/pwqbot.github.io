游戏项目实践 Project1 Domino

发表于2021-04-12|更新于2021-09-20|CS0804nGame GoUnity
|阅读量:

4.9号要交的作业, 4.7号才知道, 最后在4.12号的凌晨做完.

Love Domino

游戏项目实践课程的第一个 project, 搭建一个简单的多米诺骨牌游戏.

场景

场景

纯手工搭建的, 所以歪歪扭扭的, 用了插件 Terrain Toolbox 做一点辅助.

玩法

初始场景

玩家控制左上角的小球碰到骨牌后, 左上角的小球从左边凹槽下滑, 砸到心形左侧的骨牌.

右边的小球被抛出, 落到右边的小坡之后砸倒心形右边的骨牌.

两个小球同时下落

最后形成一个心形.

最终效果

代码

只有三个组件. 分别用来控制镜头跟随小球, 小球移动和摄像机的控制, 以及设置方块的高度和地形一致.

  • MoveCamera

  • RigidSphere

  • SetHeight
