添加脚本

先来认识一下 unity 的脚本

所有的类必须继承自 MonoBehaviour

mono-behavior是什么意思？
这表示我们可以对自己的组件进行编程，以向游戏对象添加自定义行为。这就是behavior部分所指的意思。但这是使用英式拼写就很奇怪。mono部分是指将对自定义代码的支持添加到Unity的方式。它使用了Mono项目，该项目是.NET框架的多平台实现。因此，MonoBehaviour。这是一个旧名称，由于向后兼容，我们一直持续使用。

序列化. 大概就是两个功能.

  1. 可以保存上一次的值.
  2. 可以让私有变量在面板里可视化修改.

Awake & Start

参考:

Unity中Awake和Start的区别分析

  • 不同点

    Awake function will be called at scene load if the object is set active even if the script is disabled.

    在 awake 之前所有对象都会被创建完毕. 可以把它理解成对象的构造函数.

    start 只有 gameobject 和 script 同时 active的时候才会运行.

    建议在 awake 之中进行初始化.

  • 相同点

    无论激活几次, awake 和 start 都只会运行一次.

连续移动

为了让指针移动平滑, 使用DateTime 对象 的TimeOfDay 方法. 返回一个TimeSpan 对象.

Unlike the Date property. which returns a DateTime value that represents a date without its time component, the TimeOfDay property returns a TimeSpan value that represents a DateTime value's time component.
