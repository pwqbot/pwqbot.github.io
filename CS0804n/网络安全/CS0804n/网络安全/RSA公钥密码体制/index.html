密钥生成

  1. 随机选择两个大质数 p,qp,qp,q.
  2. 计算 n=pq,ϕ(n)=(p−1)(q−1)n = pq, \phi(n) = (p -1 )(q - 1)n=pq,ϕ(n)=(p−1)(q−1)。
  3. 选择整数 e,1<e<ϕ(n)e, 1 < e < \phi(n)e,1<e<ϕ(n), 并且满足gcd(e,ϕ(n))=1gcd(e,\phi(n)) = 1gcd(e,ϕ(n))=1.
  4. 计算整数 de≡1(modϕ(n))de \equiv 1 \pmod{\phi(n})de≡1(modϕ(n)), 因为 gcd(e,ϕ(n))=1gcd(e,\phi(n)) = 1gcd(e,ϕ(n))=1,保证有解.

后面只用到 n,e,dn,e,dn,e,d 这三个数, 可以销毁p,qp,qp,q。

加密过程

加密使用公钥 (n,e)(n,e)(n,e).

ae mod n⇒ca^{e} \bmod n \Rightarrow c aemodn⇒c

解密过程

解密使用私钥(n,d)(n,d)(n,d).

cd mod n⇒ac^{d} \bmod n \Rightarrow a cdmodn⇒a

错误的证明

根据欧拉定理

aϕ(n)≡1(modn)a^{\phi(n)} \equiv 1 \pmod{n} aϕ(n)≡1(modn)

也就是说指数可以模ϕ(n)\phi(n)ϕ(n). 于是得到.

ade≡a(modn)cd≡ade≡a(modn)a^{de} \equiv a \pmod{n} \\ c^{d} \equiv a^{de} \equiv a \pmod{n} ade≡a(modn)cd≡ade≡a(modn)

其实上面的证明是错误的, 因为并不一定保证gcd(a,n)=1gcd(a,n) =1gcd(a,n)=1, 即欧拉定义不一定满足.

正确的证明

  1. gcd(a,n)=1gcd(a,n) = 1gcd(a,n)=1。

    同上

  2. gcd(a,n)≠1gcd(a,n) \neq 1gcd(a,n)=1

    根据中国剩余定理

    if(gcd(p,q)=1){x≡y(modp)x≡y(modq)⇒x≡y(modpq)if(gcd(p,q) =1)\begin{cases} x \equiv y \pmod{p} \\ x \equiv y \pmod{q} \end{cases} \Rightarrow x\equiv y \pmod{pq} if(gcd(p,q)=1){x≡y(modp)x≡y(modq)​⇒x≡y(modpq)

    可以证明一下

    x=m1p+yx=m2q+ym1p−m2q=0通解为 m1=tq带回 x=tpq+y⇔x≡y(modpq)x = m_{1}p +y \\ x = m_{2}q + y \\ m_{1}p - m_{2}q = 0 \\ \text{通解为 } m_{1} = tq \\ \text{带回 } x = tpq + y \Leftrightarrow x\equiv y \pmod{pq} x=m1​p+yx=m2​q+ym1​p−m2​q=0通解为 m1​=tq带回 x=tpq+y⇔x≡y(modpq)

    于是我们只要分别证明

    aed≡a(modp)aed≡a(modq)a^{ed} \equiv a \pmod{p} \\ a^{ed} \equiv a \pmod{q} aed≡a(modp)aed≡a(modq)

    因为gcd(a,n)≠1gcd(a,n) \neq 1gcd(a,n)=1. 且a<na < na<n. 于是aaa 要么是ppp 的倍数, 要么是qqq 的倍数.

    假设gcd(a,n)=pgcd(a,n) = pgcd(a,n)=p. 那么此时

    aed≡a(modp)a^{ed} \equiv a \pmod{p} aed≡a(modp)

    等式两边都是000, 成立.

    而此时 gcd(a,q)=1gcd(a,q) = 1gcd(a,q)=1 成立. 所以有

    aq−1≡1(modq)a^{q - 1} \equiv 1 \pmod{q} aq−1≡1(modq)

    ade≡akϕ(n)+1≡ak(p−1)(q−1)+1≡a(modq)a^{de} \equiv a^{k\phi(n) + 1} \equiv a^{k(p-1)(q-1) + 1} \\ \equiv a \pmod{q} ade≡akϕ(n)+1≡ak(p−1)(q−1)+1≡a(modq)

    which proves the theorem.
